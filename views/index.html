Extra-Curricular Courses I Have Taken

AI Programming with Python Nanodegree

By: Udacity
Completion Date: 2021

The AI Programming with Python Nanodegree aims to teach students the fundamentals of machine learning and AI using Python. It starts with their mathematical basis, such as Linear Algebra and Calculus, before introducing often used libraries such as NumPy, Pandas, Matplotlib and PyTorch. Lastly, students are taught how neural networks work, as well as how to train and optimize them.

I took up this course in my second year of University as I wanted to venture into a different field. AI / Machine Learning has always fascinated me as a concept, but at the time I had little to no knowledge as to what it was, let alone how they worked. This course served as my introduction to the world of AI, and also reinforced my proficiency in Python.

Skills: Python, Keras, PyTorch, Neural Networks, Optimizers, NumPy, Pandas, AI, Machine Learning

The Web Developer Bootcamp

By: Udemy
Completion Date: 2018

The Web Developer Bootcamp is a highly rated full-stack web development course lead by instructor Colt Steele. In this course, students are introduced to the MERN tech stack, as well as learning how to develop fundamental web-page functionalities, such as user authentication, routing, responsive page elements etc., through multiple side projects.

Web Development was the very first field of CS that I explored, and this course served as my entry point. Full-Stack development is no easy task, as managing both front-end and back-end systems requires strong knowledge of the overall web-page's architecture. To tackle this issue, this course not only taught me modern frameworks used in web development such as React, but also introduced to me useful programs and shortcuts to aid in the development process, such as NPM.

Skills: HTML5, CSS, Javascript, Express, MongoDB, React, Node.js, Full-Stack Development

CS50: Introduction to Computer Science

By: Harvard
Completion Date: 2017

CS50 was conceived as an online introductory course for prospective computer science students for both Harvard and Yale, lead by Harvard professor David Malan. It introduced basic concepts like data structures and algorithms, using languages such as C and Python.

This challenging program was my first foray into programming. It paved a solid foundation for my CS journey and allowed me to pick up new skills / technologies quickly as a result.

Skills: C, Python, Algorithm Optimization
